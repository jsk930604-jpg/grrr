KIS Swing Scoring System

한국투자증권 시세 데이터를 기반으로 스윙 종목을 점수화해 상위 후보를 추리는 자동 분석 도구입니다. 아래 가이드는 설치, 실행, 스케줄 운영, 점수 해석까지 한 번에 볼 수 있게 정리했습니다.

Top 10 랭킹 ETF/ETN 자동 제외 1,000원 이하 제외 투자주의 제외 장마감 1일 1회

1) 데이터 수집

거래량/등락률 랭킹 + 지수 + 기관수급 + 종목 차트를 합쳐 분석 대상 풀을 만듭니다.

2) 필터링

ETF/ETN, 1,000원 이하, 투자주의 종목을 우선 제거해 노이즈를 줄입니다.

3) 점수화

추세/모멘텀/거래량/변동성/시장동조 합산(100점)으로 후보 우선순위를 정합니다.

빠른 시작 가이드

Step 1. 의존성 설치 
npm install
Step 2. 환경변수 준비

.env.example를 참고해 .env에 한국투자증권 API 키를 입력합니다.

Step 3. 즉시 1회 실행 테스트 
npm run start:once
Step 4. 운영 모드(스케줄 대기) 
npm start

기본 설정은 평일 장마감 시간대 1회 자동 실행입니다.

점수 기준 (100점)

추세30
모멘텀25
거래량20
변동성15
시장동조10

해석: 80점 이상 강매수, 70~79점 매수후보, 70점 미만 관망

운영 팁

후보 풀이 너무 적을 때

KIS_CANDIDATE_LIMIT를 80~120으로 늘려 탐색 범위를 넓히세요.

너무 급등주가 많이 잡힐 때

KIS_MIN_PRICE를 올리고, 결과에서 변동성 점수 비중을 함께 확인하세요.

장중 부하를 줄이고 싶을 때

KIS_CONCURRENCY를 낮춰 API 동시호출량을 조절하세요.
